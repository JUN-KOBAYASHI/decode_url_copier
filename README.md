# Decoded URL Copier

現在のタブのURLをデコード（日本語表記）してクリップボードにコピーする、シンプルなGoogle Chrome拡張機能です。

WikipediaのURLや日本語検索結果のURLなど、通常は `%E3%81%82...` のようにエンコードされて長くなってしまうURLを、人間が読みやすい形式（例: `https://ja.wikipedia.org/wiki/日本語`）でコピーできます。

## 特徴

* **ワンクリック操作**: 拡張機能アイコンをクリックするだけで動作します。
* **日本語対応**: URLエンコードされた日本語文字列をデコードします。
* **自動クローズ**: コピーが完了すると、ポップアップは自動的に閉じます。
* **安全設計**: `activeTab` 権限のみを使用し、バックグラウンドでの常駐プロセスはありません。

## インストール方法 (開発者モード)

Chromeの「デベロッパーモード」を使用してインストールします。

1. このリポジトリをクローンするか、ZIPでダウンロードして解凍します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力します。
3. 画面右上の **「デベロッパーモード」** をオンにします。
4. 左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. 解凍したフォルダ（`manifest.json` が含まれているフォルダ）を選択します。

## 使い方

1. コピーしたいWebページを開きます。
2. Chromeのツールバーにある **「Decoded URL Copier」アイコン**（リンクのマーク）をクリックします。
3. 「コピーしました！」と表示されればクリップボードへのコピー完了です。

## ファイル構成

* `manifest.json`: 拡張機能の設定ファイル (Manifest V3準拠)
* `popup.html`: ポップアップウィンドウのUI
* `popup.js`: URL取得とコピー処理を行うスクリプト
* `icons/`: アイコン画像ファイル
    * `icon16.png`
    * `icon32.png`
    * `icon48.png`
    * `icon128.png`
